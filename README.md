## 🚀 Overview

FFMPEG Log Analyzer is a simple Web-Based Log-Analyzer for FFMPEG.

## 🎯 Features

- Show max Speed
- Show max FPS

## 💻 Usage

Head over to my instance on [tools.massimo.gg](https://tools.massimo.gg/ffmpeg/), upload your FFMPEG log and see the results instantly.
